五、第一篇博客

在搭建博客的过程中一定会遇到各种报错和各种尝试，记下来之后不妨直接把它作为第一篇博客。我使用的是Typora+Idea，使用Idea将MyBlog作为Project打开，进行管理。Typora就用来专心码字。在还没学图床及CDN加速的基础上，所以这篇教程只有文字，如果有不懂的地方可以多用Google搜索一下其他的教程。

找到~/MyBlog/source/_posts/hello-world.md，里面放的就是你在自己的博客网站上看的内容。删除hello-world.md，将自己的笔记放进去。

关于Front-matter：

在Typora中写博客的时候，需要手动将YAML信息放在文章顶部，键入---，Typora会自动显示，请输入YAML信息。

其中最基础的应该就是title和date，注意冒号后面必须要有空格。

十一、替换网站图标

自己制作或者下载16 x 16px以及32 x 32px的PNG格式图标。将图片放在MyBlog/themes/next/source/images/下。

打开MyBlog/themes/next/_config.yml，查找favicon，将small和medium后的favicon-16x16-next替换为对应像素的图片的名称。
